# Projeto de Automação e Análise de Dados com Python e Envio Automatizado de E-mails

### 1️⃣ Descrição do Projeto
Este projeto foi desenvolvido para automatizar uma tarefa rotineira da área de análise: acessar uma base de vendas disponibilizada em um sistema, baixar o arquivo, calcular os principais indicadores do dia e enviar automaticamente um e-mail com o relatório para a diretoria.
A solução foi construída em Python, utilizando automação de interface com pyautogui e análise de dados com pandas.

### 2️⃣ Objetivo
Automatizar o processo diário de envio de relatório de vendas, reduzindo atividades manuais, economizando tempo e minimizando erros operacionais.

### 3️⃣ Funcionalidades

  - Abertura automática do navegador
  - Acesso ao sistema/Google Drive
  - Leitura do arquivo Excel com Python
  - Cálculo de indicadores de negócio
  - Abertura do Gmail
  - Preenchimento automático do destinatário, assunto e corpo do e-mail
  - Envio automático do relatório
  - 
### 4️⃣ Indicadores calculados

  - Faturamento total
  - Quantidade total de produtos vendidos

### 5️⃣ Tecnologias e bibliotecas utilizadas

  - Python
  - PyAutoGUI — automação de mouse e teclado
  - Pandas — leitura e análise dos dados
  - Openpyxl — leitura de arquivos .xlsx
  - Pyperclip — cópia e colagem de textos
  - Time — controle de pausas durante a automação
    
### 6️⃣ Aprendizados com o projeto

  - Automação de processos com Python
  - Integração entre automação e análise de dados
  - Manipulação de planilhas Excel
  - Geração de indicadores para apoio à tomada de decisão
  - Automatização de rotinas administrativas e operacionais
